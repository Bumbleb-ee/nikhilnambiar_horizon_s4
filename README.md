
Team Horizon CUSAT Recruitment Tasks - S4

## Repository 
- `level1/` - Rover Coordinates and Motion Calculator (Python)
- `level2/` - Arduino Servo Control Simulation (Tinkercad)
- `src/my_robot_controller/` - ROS2 Publisher, Subscriber and Decision Nodes
- `screenshots/` - Proof of working simulations

## Level 1 - Rover Coordinates
Python program to calculate distance and travel time between two coordinates with motion parameters and error handling.

## Level 2 - Arduino Servo Control
Tinkercad simulation of potentiometer controlling servo motor limited to 68 degrees with LED warning indicator.

## Level 3 - ROS2 Nodes
ROS2 Humble publisher/subscriber nodes simulating rover distance sensing and autonomous movement decisions.

## Environment
- Python 3.12
- Arduino (Tinkercad Simulation)
- Ubuntu 22.04
- ROS2 Humble
